# qfilib

**Quantitative Fixed Income Library**

A Python library for quantitative analysis of fixed income instruments.

## Project Structure

- `src/` - Core library modules for instruments, backtesting, auction modeling, and quantitative research
- `notebooks/` - Jupyter notebooks for analysis and research
- `notebooks_wip/` - Work-in-progress notebooks
- `data/` - Data files and datasets

## Key Files

- `config.py` - Configuration settings for the library
- `daily_dag.py` - Daily workflow and task orchestration

## Requirements

- Python 3.12+
- T
